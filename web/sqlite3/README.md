# SQLite para Web

Este diretório contém os binários necessários para o SQLite funcionar no ambiente web.

Os arquivos necessários são:
- `sqlite3.wasm` - O binário WebAssembly do SQLite
- `sqlite3.js` - O JavaScript API para SQLite

Estes arquivos são necessários para que o pacote `sqflite_common_ffi_web` funcione corretamente.
